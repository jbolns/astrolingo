**AstroLingo** es una plantilla/tema Astro que busca facilitar el diseño de sitios web personales, blogs y portafolios multilingües.

Está basado en la [plantilla de blog minimalista](https://astro.build/themes/details/blog/) de Astro.

## Cuando es buena idea usar *AstroLingo*?
Usar *AstroLingo* para sitios web en un solo idioma es innecesario. Por definición, un sitio multilingüe requiere de código un poco más complicado que un sitio en un sólo idioma.

Ahora bien, si necesitas o quieres un sitio web personal, blog o portafolio multilingüe, *AstroLingo* podría ahorrarte muchas horas de trabajo; días, incluso.

## Documentación
Para mantener el código lo más simple posible, **AstroLingo** maximiza la estructura estándar y la funcionalidad estándar de Astro.

Si comprendes la estructura de un proyecto Astro típico, es probable que AstroLingo te resulte fácil de entender.

Para más detalles de como funciona la cosa, leete el repositorio de GitHub de [*AstroLingo*](https://github.com/jbolns/astrolingo).
