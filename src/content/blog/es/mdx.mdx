Este tema viene con la integración [@astrojs/mdx](https://docs.astro.build/en/guides/integrations-guide/mdx/) instalada y configurada en tu archivo de configuración `astro.config.mjs`. Si prefieres no usar MDX, puedes deshabilitar MDX removiendo la integración del archivo de configuración.

## Por qué MDX?

MDX es una variante especial de Markdown que permite utilizar JavaScript & JSX syntax. Esto permite [mezclar componentes JavaScript y UI en tu contenido Markdown](https://docs.astro.build/en/guides/markdown-content/#mdx-features) para cosas como gráficos interactivos o alertas.

Si tienes contenido que ya has hecho en MDX, esta integración te puede ayudar a migrar a Astro en un dos por tres.

## Ejemplo

Aquí abajo hay un ejemplo de como importar y usar un componente adentro de MDX.  
Cuando abres esta página en un navegador, deberías ver un botón al que le puedes dar click.

import HeaderLink from '@components/HeaderLink.astro';

<HeaderLink href="#" onclick="alert('clicked!')">
	Componente incorporado adentro de MDX
</HeaderLink>
